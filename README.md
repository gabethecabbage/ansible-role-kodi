# Ansible role: Kodi

Ansible role for installing Kodi media server
Forked from dragomirr.kodi

## Requirements

Setting ansible python interpreter to python3.

## Dependencies

None.

## Example playbook

```
- hosts: all
  roles:
    - role: gabethecabbage.kodi
      ansible_python_interpreter: /usr/bin/python3
```
